# Linear Regression Model

## Problem Statement
Develop a simple linear regression model for predictive analysis. Evaluate model performance using metrics like MSE and R2.

## Technologies Used
- Python 3.x
- scikit-learn
- matplotlib

## How to Run
1. Clone the repo
2. Run `python main.py`

## Sample Output
- MSE: <1.86>
- R2: <0.07>
